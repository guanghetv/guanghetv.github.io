## 用法

1. 下载、解压文件，将解压完的文件夹移动到项目文件夹
2. 在命令行中移动到所在目录
3. 执行`npm install`
4. 执行'npm start start-dev'
5. 打开浏览器并访问[http://localhost:3000][3]
