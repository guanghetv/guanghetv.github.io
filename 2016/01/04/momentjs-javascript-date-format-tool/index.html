Moment.js - JavaScript的时间解析、校验、操作、展示工具。

常用格式：

  • 年：YYYY(2016)|YY(16)|GGGG(2016)|GG(16)|gggg(2016)|gg(16)
  • 月：M(1)|MM(01)|MMM(Jan)|MMMM(January)
  • 日：D(1)|DD(01)|Do(1st)
  • 时：h(1)|hh(01)|H(13)|HH(13)
  • 分：m(1)|mm(01)
  • 秒：s(1)|ss(01)
  • 毫秒：S(1)|SS(01)|SSS(001)
  • 上下午：a(am)|A(AM)
  • 星期：dd(mon)|ddd(monday)|e(1)|E(1)
  • 周数：W(1)|WW(01)
  • 天数：DDD(1)|DDDD(001)
